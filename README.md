## 🎨 Design System

### Color Palette
| Token | Value | Usage |
|---|---|---|
| Primary | `#5B2BFF` (Electric Violet) | CTAs, highlights |
| Secondary | `#00D4FF` (Cyan Blue) | Accents, AI theme |
| Dark BG | `#0A0A0F` | Page background |
| Dark Surface | `#13131A` | Cards, panels |
| Text Primary | `#FFFFFF` | Headings |
| Text Muted | `#A0A0B0` | Descriptions |
| Gradient | Violet → Cyan | Hero, banners |

### Typography
- **Heading Font**: `Outfit` (Google Fonts)
- **Body Font**: `Inter` (Google Fonts)

### Key UI Components
- Glassmorphism cards
- Neon glow effects on hover
- Smooth scroll + fade-in animations
- Sticky navigation with blur backdrop
- Mobile-first responsive grid
